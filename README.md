# 🌍 Proyecto Basura Cero · ODS 13

Aplicación CRUD para gestión de residuos, desarrollada como parte del proyecto de clase para concienciar sobre la **Acción por el Clima (ODS 13)**.

---

## 🚀 Características

- Registro de residuos con múltiples campos (nombre, color, peso, material, estado, etc.)
- Listado dinámico con filtro por nombre
- Edición y eliminación de registros
- Persistencia de datos en `localStorage`
- Interfaz de dos vistas integradas en una sola página

---

## 🛠️ Tecnologías

- HTML5, CSS3, JavaScript (ES6)
- Git & GitHub para control de versiones
